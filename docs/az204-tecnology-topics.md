# Tópicos Granulares para Simulado AZ-204 (2026)

## 1. Soluções de Computação
* **Azure App Service**: Configuração, planos de serviço e slots de implantação.
* **Azure Functions**: Gatilhos (triggers), vínculos (bindings) e planos de consumo.
* **Azure Container Registry (ACR)**: Gerenciamento e publicação de imagens.
* **Azure Container Instances (ACI)**: Execução de contêineres isolados e grupos de contêineres.
* **Azure Container Apps (ACA)**: Microsserviços conteinerizados e escalonamento baseado em KEDA.

## 2. Armazenamento de Dados
* **Azure Blob Storage**: Gerenciamento de containers, tiers de acesso e ciclo de vida.
* **Azure Cosmos DB**: APIs, níveis de consistência e particionamento de dados.

## 3. Segurança e Identidade
* **Microsoft Entra ID**: Fluxos de autenticação, autorização e tokens.
* **MSAL (Microsoft Authentication Library)**: Implementação de login em código .NET/Java/Python.
* **Shared Access Signatures (SAS)**: Criação de chaves de acesso limitado e políticas de acesso.
* **Azure Key Vault**: Gestão de segredos, certificados e chaves criptográficas.
* **Managed Identities**: Identidades atribuídas pelo sistema vs. atribuídas pelo usuário.
* **Azure App Configuration**: Gerenciamento centralizado de flags de funcionalidade e configurações.

## 4. Integração: Mensageria e Eventos
* **Azure Event Grid**: Arquitetura baseada em eventos e tópicos de sistema/personalizados.
* **Azure Event Hubs**: Ingestão de telemetria em larga escala e streaming.
* **Azure Service Bus**: Filas (Queues) e Tópicos/Assinaturas (Topics/Subscriptions) com garantia de entrega.
* **Azure Queue Storage**: Filas simples para grandes volumes de mensagens.

## 5. Integração: APIs
* **Azure API Management (APIM)**: Políticas (policies), backends e portal do desenvolvedor.
* **Microsoft Graph**: Consulta de dados de usuários e recursos via API REST.

## 6. Monitoramento e Otimização
* **Application Insights**: Telemetria, log de exceções e mapeamento de dependências.
* **Azure Cache for Redis**: Implementação de cache distribuído e estratégias de expiração.
* **Azure CDN (Content Delivery Network)**: Cache de borda e otimização de entrega global.
